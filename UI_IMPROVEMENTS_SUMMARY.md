## 🎨 КАТЕГОРІЇ - Покращення

### Зараз:
- Donut pie chart з категоріями
- Список категорій з іконками, кольорами, сумами
- Лінійний чарт щоденних витрат для обраної категорії
- Список транзакцій з фільтрами
- Вибір періоду (пресети + календар)
- Створення власних категорій

**Проблеми:**
1. Відсутній контекст бюджету - не видно скільки залишилось

3. Тільки один тип візуалізації (pie)
4. Великі списки важко скролити
5. Базові empty states

### Технічна реалізація:

**Формули:**
```typescript
dailyBurnRate = totalSpent / daysPassed;
```
